## The Theme I used

Dante - Astro & Tailwind CSS Theme by justgoodui.com

Dante is a single-author blog and portfolio theme for Astro.js. Featuring a minimal, slick, responsive and content-focused design.

For more Astro.js themes please check [justgoodui.com](https://justgoodui.com/).

Thank you [astab](https://github.com/astab) (and the other contributors) for this theme.
